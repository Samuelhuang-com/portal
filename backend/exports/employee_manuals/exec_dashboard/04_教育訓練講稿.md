## 功能用途說明

高階主管 Dashboard 是集團最高層級的決策資訊中心，整合飯店管理、商場管理、保全巡檢、工務報修等各大模組的核心數字，讓董事長、總經理等高階主管在一個畫面內掌握全集團營運狀況，不需要逐一切換不同功能頁面。

（重點說明）
這個功能主要是給「董事長 / 總經理 / 副總經理 / 高階主管」使用的，
大家在日常工作中如果需要早晨快速確認昨日整體狀況，
就可以打開這個頁面。

**進入位置：** 登入後，點選 Dashboard > 高階主管 Dashboard

## 常見錯誤提醒

（特別叮嚀以下容易出錯的地方）

- **畫面顯示轉圈很久**：等候 30 秒後重新整理頁面，若持續請聯絡管理員
- **數字顯示為 0 或空白**：確認月份正確，聯絡管理員確認同步狀態
- **看到「無權限」提示**：聯絡系統管理員申請開通高階主管 Dashboard 瀏覽權限
- **TAB 點了沒有反應**：等待畫面完整載入後再點選，建議使用 Chrome 瀏覽器
