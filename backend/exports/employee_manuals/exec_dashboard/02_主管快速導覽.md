# 主管快速導覽：高階主管 Dashboard

## 功能簡介

高階主管 Dashboard 是集團最高層級的決策資訊中心，整合飯店管理、商場管理、保全巡檢、工務報修等各大模組的核心數字，讓董事長、總經理等高階主管在一個畫面內掌握全集團營運狀況，不需要逐一切換不同功能頁面。

**選單位置：** Dashboard > 高階主管 Dashboard

---

## 核心 KPI 一覽

主管最需要關注的數字：

| KPI 名稱 | 代表意思 | 數字偏高時要注意 |
|---|---|---|
| 整體完成率 | 全集團所有待辦事項的整體完成百分比 | 整體運作狀況良好 |
| 飯店工務完成率 | 飯店端工務相關工作的完成百分比 | 飯店工務執行力強 |
| 商場工務完成率 | 商場端工務相關工作的完成百分比 | 商場工務執行力強 |
| 異常件數 | 本月各模組登記為異常的總件數 | 需特別關注，了解異常原因 |

---

## 主管常用操作情境

- **早晨快速確認昨日整體狀況**：登入 → 查看「總覽」TAB → 確認各模組 KPI 是否正常
- **主持週會需要數字報告**：切換「趨勢」TAB → 查看過去幾週的完成率變化
- **發現某模組完成率偏低**：點選對應 TAB（如「飯店」）→ 確認哪個子模組異常 → 聯絡該部門主管
- **想了解本月異常件數原因**：切換「報修」或「保全」TAB → 查看異常詳情

---

## 管理重點提醒

- 此 Dashboard 的數字每天自動更新，不是即時資料
- 若有緊急狀況需確認最新進度，請直接進入各模組頁面查看
- 「趨勢」TAB 顯示的是過去 6 個月的月統計數字
- 此頁面僅開放給被授予高階主管 Dashboard 權限的帳號

## 判斷標準建議

| 狀態 | 建議行動 |
|---|---|
| 完成率 ≥ 90% | 正常，繼續維持 |
| 完成率 70%~89% | 注意追蹤，了解落後原因 |
| 完成率 < 70% | 立即介入，召集相關人員討論 |
| 異常件數 > 上月 | 深入了解異常原因，確認是否有系統性問題 |

---

## 快速行動清單（每日）

1. 查看整體完成率是否正常
2. 確認異常件數是否增加
3. 若有待處理件數偏高，追蹤負責人員處理進度
4. 月底前確認所有工作項目達標
