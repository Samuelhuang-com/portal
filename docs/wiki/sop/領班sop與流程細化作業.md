# 領班SOP與流程細化作業

- 飯店：英迪格
- 部門：房務部
- 日期：2025/9/29

**版本**：v0.1

**摘要**：

- 本文件提供樓層領班在 09:00–17:00
  的標準作業指引，將「檢查、清理、維修、溝通、回報」轉化為可執行的工作清單、量化檢核與驗收方式。

- 適用對象為各樓層領班與房務組，跨部門(FO/ENG/SEC)通報格式一併納入。

- 補強「領班必檢必做」與「單一房間查房作業」；新增拆/併床通知流程、擦塵標準、即期備品與冰箱/迷你吧效期稽核、排水與馬桶蓋拆洗等高風險細節與量化驗收。

- 往後遇到客訴，將檢討此SOP，並進行記錄、改版修正。
## 1. 領班「必檢必做」清單
- **拆/併床**：

  - **流程**：

    - 預報需求 → FO 確認 → HK 變更床組(床橋／防滑墊／床單規格) →
      照片驗收 → PMS「房型／鋪床型態」同步 →
      交班標註是否需還原(若次日非同型)。

  - **操作要點**：

    - 拆／併前斷電保護、鎖定床輪，清出作業區 60cm 以上。

    - 併床：置床橋＋連接帶固定、防滑墊到位、加保潔墊後再鋪床；拆床：解除連接帶→撤床橋→復位防滑。

    - 床單規格比對房型(King／Twin)，被套、枕套同步校對。

  - **檢核／驗收**：接縫高差 ≤［\_\_
    mm］、躺臥不位移、四角服貼；完成**前後照**(全景／接縫近拍)，PMS
    欄位與照片路徑齊全。

  - **房型對照表：需經常性更新實際房型**

![房型對照表-房型說明](media/image2.png)

- **除濕機(走道／客房)**：

  - 每日巡檢電源與運轉聲；檢查水箱水位／連續排水是否正常，**滿水即倒水或恢復連排**；

  - 設定目標濕度［\_\_%RH］並**啟用節能/定時模式**；

  - 確認進出風口無遮擋
    ≥［\_\_cm］、線材無滲水受潮；異常(噪音、異味、無除濕)→
    立即報修(ENG)並標示「維護中」。

- **擦塵標準**：

  - **順序**：由高至低、由內向外；相框邊、出風口百葉、燈具、電視背、床頭板上沿、踢腳線、底部線槽。

  - **檢核**：以**斜側光**檢視無灰帶、無手印、無積塵邊；手套擦拭不留灰痕。

  - **不合格處置**：現場復掃→再以斜側光複檢；連續二次不合格登錄教育訓練名單。

- **即期備品/食品(FEFO)**：

  - **範圍**：飲料、咖啡膠囊、茶包、衛浴耗材(含補充罐)。

  - **規範**：效期 ≤［\_\_
    天］即下架；採\*\*先到期先出(FEFO)\*\*補位；每批次留存「批號／到期日」。

  - **檢核**：抽查比率［\_\_％］／日，過期率＝0；下架品記錄退倉或報廢去向。

- **冰箱/迷你吧**：

  - **檢查點**：效期、數量、溫度［**–**°C］、封條完整、冰箱燈與風道暢通。

  - **操作**：不足即補；斷電保冷要**交班註記**並張貼提示；封條破損當場更換。

  - **檢核**：溫度達標、數量與價單一致、封條無破損；拍照留存異常。

- **排水管/地漏**：

  - **操作**：淋浴地漏**開蓋清髒**、洗手台／浴缸清除落髮；注水測流速。

  - **檢核**：洗手台自滿水線至排空 ≤［\_\_
    秒］；淋浴區排水無倒灌、無異味回湧；必要時補注存水彎。

  - **不合格處置**：化學或機械通管，仍不通→ENG 報修並標記 OOO。

- **馬桶蓋拆卸擦拭**：

  - **操作**：掀蓋露出鉸鏈→拆下座圈與蓋板→**清洗鉸鏈座、座圈底邊與螺帽區**→乾燥後復位鎖緊。

  - **檢核**：座圈不晃動、對正不偏移；沖水水封回位、無滲漏與異味；外沿無水痕。

  - **驗收**：近拍鉸鏈區清潔前／後照；異常立刻報修。

- **浴室玻璃清潔程序**：

  - **步驟：**熱水軟化水垢 → 噴中性清潔劑 → 先擦磁磚牆面再擦玻璃 →
    玻璃刮刀(或抹布)自上而下 → 乾布收邊不留水紋。

  - **檢核：**逆光觀察無水紋、無手印、無清潔劑殘留；五金件無水垢圈。

- **床底/家具下方**：

  - **檢查**：以手電(或
    UV)檢視雜物／落塵／蟲跡；吸塵器**延伸頭**深入清除邊角與走線槽。

  - **檢核**：無雜物、無灰帶、無蟲跡；發現異常即拍照並通報防治。

- **電視頻道與連網檢查(含房號識別)**：

  - **操作**：

    - 測試遙控器所有主要按鍵(電源／數字鍵／音量／靜音／訊號源)，確認**即時響應**。

    - 依清單巡檢頻道［抽測
      ≥［\_\_］台］，含：飯店資訊頻道、新聞、體育、電影、地方台；機上盒／HDMI
      來源切換正常。

    - 檢查智慧連網：Wi-Fi 連線成功、內建 OTT(如 YouTube)開啟並試播［\_\_
      分鐘］；可選測速≥［\_\_ Mbps］。

    - 開機預設與**房號識別**：顯示歡迎頁／資訊欄之**房號**與 PMS
      一致；重啟後能回到預設頻道。  
      **檢核**：遙控器按鍵**無延遲或漏按**(反應 ≤［\_\_
      秒］)；頻道畫面與音訊穩定無雪花／無雜音；串流影片連續播放無中斷；房號顯示正確。

  - **驗收**：

    - 必要時拍攝螢幕照(頻道、歡迎頁／房號)；異常詳記「來源(TV/機上盒/網路)＋處置」。

  - **異常處置**：

    - 無訊號／黑畫面→檢查訊號源與 HDMI
      線、重啟機上盒；仍故障→報修(ENG)並標記房態「待修」。

    - 遙控器不靈→更換電池／配對；

    - 房號顯示錯誤／歡迎頁未更新→通知 FO／系統管理修正 CMS／PMS
      對應欄位。

  - **系統更新**：在「設備檢查」欄填寫結果；若報修，請通知工務ENG。

- **浴室吹風機(壁掛／抽屜式)檢查**：

  - **操作**：

    - 檢視外觀與絕緣：機身／電線無破損、**防滴水走線**良好；壁掛鉤／座固定**不鬆動**。

    - 通電試轉：低／中／高檔風速皆能穩定運轉；熱風檔出風溫度達標(約［**°C］±［**］)；過熱保護可正常作動。

    - 濾網清潔：打開進風口檢查棉絮，必要時計刷或更換濾網。

  - **檢核**：掛鉤承重手拉測試(≥［\_\_
    kg］)不位移；各檔風速有明顯差異、無異音／焦味；電源鍵與安全開關回彈良好。

  - **驗收**：拍照(掛座固定點／機身序號／運轉測試)；清潔後復位；張貼使用注意貼紙清晰可讀。

  - **異常處置**：

    - 掛鉤鬆動／位移→**立即停用**、張貼「維護中」、報修(ENG)加固；

    - 風量不足／過熱跳脫→更換或報修；

    - 有入水風險(濺水區)→**即刻停用**並改善線路與防滴水導向。

  - **系統更新**：在「設備檢查」欄記錄檔位測試結果與濾網清潔時間；若報修填列工單號與時間戳。

- **安全與功能抽測**：

  - **項目**：門鎖反鎖／貓眼、窗扣／限位、保險箱歸零、吹風機／快煮壺、遙控器電量、插座鬆動、緊急疏散圖與消防設備无遮挡。

  - **檢核**：抽測點全通過；任一不通過→立即\*\*報修(ENG)\*\*並於房態標註「待修」或
    OOO。

- **照片佐證規範**：

  - **命名**：YYYYMMDD_房號_項目_前／後。  
    **必拍觸發**：拆／併床完成、工程驗收、異味／髒污**前後**、DND
    通知投遞。

  - **補充**：需含一張**全景**與一張**重點近拍**；上傳時限：完成後［\_\_
    分鐘］內。

- **系統欄位更新**：

  - **必填**：房態、鋪床型態、是否預進、是否
    OOO、缺失工單號、照片路徑；**皆需時間戳與責任人**。

  - **檢核**：欄位完整率 100%，與實況一致率
    ≥［\_\_％］；差異即回溯更正並註記原因。
## 2. 單一房間查房作業(依房況分流)
領班在每一種房況下的**可量化檢核**與**驗收**；所有時限可依樓層填入。
### 2.1 空房(Vacant/Clean)終檢 → 釋放可售
- **目標**：最終確認可售。**時限**：每間［**–** 分］。

- **步驟與檢核**

  1.  **環境與安全**：門鎖、貓眼、門縫止條完整；窗扣/限位正常(抽測)；地面無水痕。

      - ✅ 判定：抽測 2 點通過；鬆動/異常 → 報修(ENG)SLA［\_\_ 分］。

  2.  **擦塵**：高→低：燈具/相框邊/出風口百葉/電視背/床頭上沿/踢腳線；**斜側光檢查**。

      - ✅ 無指痕與灰帶；有灰帶→原地修復＋抽查加倍。

      - ✅ 房間內玻璃杯檢查，於亮光處檢查有無水痕、污漬

  3.  **床組**：床單尺寸對型(King/Twin)、邊角 45° 服貼、枕被蓬鬆、置中。

      - ✅ 接縫平整、無毛髮與污漬；錯規格→立即更換並回報管洗。

      - ✅ 已整好的棉被，也要掀起檢查有無毛髮。

  4.  **備品/迷你吧/冰箱**：數量/效期/封條；冰箱溫度 ［**–**°C］。

      - ✅ 效期 ≥［\_\_天］；不足或過期→下架記錄＋補貨。

  5.  **衛浴**：玻璃(**熱水＋清潔劑＋刮刀**)、檯面、鏡面、金屬件無水痕；排水測試；馬桶蓋**拆洗**後裝回。

      - ✅ 玻璃無水紋；落髮≦1 根/㎡；排水通暢；馬桶座圈無污。

  6.  **設備抽測**：空調(21°)、照明、插座、吹風機、電熱壺、電視換台。

      - ✅ 全通過；異常→工單＋房態標註。

  7.  **底部清潔**：床底/家具下方、垃圾桶內外沿。

      - ✅ 無雜物與灰塵帶。

      - ✅ 書桌下方不要安置垃圾袋，浴室則需要。

  8.  **照片與系統**：必要項目拍照；房態改「空房(可售)」；紀錄完成時間。

- **常見異常處置**：異味→臭氧(10分鐘)［\_\_ 分］＋通風［\_\_
  分］再複檢；插座鬆動→ENG 緊急(高)。
### 2.2 續住(Stayover)關懷檢查
- **目標**：舒適與衛生維持。**時限**：每間［**–** 分］。

- **步驟與檢核**

  1.  垃圾分類清運，補水/備品(按房型)；**即期備品**再確認。

  2.  快速除塵＋地面吸塵(床邊/桌角/踢腳)；玻璃手印點拭。

  3.  浴室：髒點擦拭、馬桶外沿與座圈上下面；排水快速測試。

  4.  設備：空調操作詢問、遙控器功能。

  5.  若客人在房：**徵詢偏好**(枕數、清潔時段)並記錄。

- **驗收**：補件照片(如有異常)；系統備註「續住偏好」。

- **異常**：客訴/過敏→立即升級處理與回訪。
### 2.3 退房(Check-out)終檢
- **目標**：確認清潔完成並釋放可售(預進優先)。**時限**：每間［**–**
  分］。

- **步驟與檢核**(抽查比例［\_\_％］)

  - 與 B1
    相同檢核；**增加**：保險箱清空歸零、迷你吧盤點、遺失物檢查(床下/抽屜/浴簾後)。

- **驗收**：合格→改「空房」並通報 FO；不合格→返工＋再抽。
### 2.4 DND(Do Not Disturb)程序
- **目標**：尊重隱私與安全。

- **步驟**

  1.  首次發現登錄時間；

  2.  滿［\_\_ 小時］：**二次關懷**(電話＋門縫通知卡)；

  3.  滿［\_\_ 小時］仍無回覆且有風險→**SEC 安全關懷**。

- **驗收**：每次關懷留紀錄/拍照；必要時事件報告。
### 2.5 OOO(工程房)與解除流程
- **目標**：工程房不上架，可售時迅速復歸。

- **步驟**

  1.  標記 OOO＋開工單；

  2.  完工驗收(ENG+HK)：功能測試清單全通過；

  3.  改「髒房」→清潔→改「空房」；

  4.  系統填入工單號與照片。

- **時限 KPI**：解除 OOO 至可售 ≤［\_\_ 分］。
### 2.6 拆/併床標準流程(領班必管)
- **觸發**：訂單要求或調房需要(King ↔ Twin)。

- **工具**：床橋/床墊連接帶、防滑墊、保潔墊(對應尺寸)、King/Twin
  床單組。

- **步驟**

  1.  **FO 確認**需求與下次住客型態；

  2.  **HK 執行**：移位→安裝床橋/連接帶→鋪保潔墊→鋪床；

  3.  **驗收**：接縫不高於［\_\_ mm］、不位移；

  4.  **系統**：更新「鋪床型態」欄位；上傳完工照；

  5.  **交班**：下一筆預進非同型→**還原排程**。

- **安全**：鎖定床輪；避免壓線與夾手。
## 3. 領班日常流程
### 3.1 09:00–09:10｜交班核對(領班)
- **目標**：取得即時房態、異常清單與優先序。

- **步驟**(優先：高→低｜耗時)

  1.  取得「房態對帳表」(可售/在住/髒房/OOO/DND/預進)—(3m)

  2.  接收「異常待辦」：工程工單、客訴、超時DND、VIP/預進房—(4m)

  3.  確認人力名冊與請假異動—(3m)

  4.  取得房態/異常清單；**點名床型變更(拆/併床)**、預進重點。

  5.  驗收：交接簿簽名＋系統截圖。

- **工具/表單**：交接簿、房態報表、工單系統(OneTouch／PMS／ENG)。

- **負責角色**：主要—領班；次要—夜班代理／前台值班。

- **可量化檢核**：交接項目完整率＝100%；房態一致率≥［\_\_％］。

- **驗收方法**：交接簿簽名＋系統截圖存檔(命名：YYYYMMDD_交接)。

- **常見異常與處置**：

  - 工程房未在系統標 OOO→立即致電ENG並開單；回應時限［\_\_分］。

  - 預進名單與可售不符→通知FO調整，回覆時限［\_\_分］。
### 3.2 09:10–09:25｜班前佈署與工具點檢
- **目標**：完成任務分配與工具就緒。

- **步驟**

  1.  任務派發：預進房優先、退房帶節點分配—(7m｜高)

  2.  推車/備品/臭氧機/抽水機點檢(數量、電量)—(5m｜高)［待合併附件：B1
      推車調度］

  3.  宣達今日KPI(釋房時點、抽查比例)—(3m｜中)

  4.  驗收：點檢表＋缺失工單。

- **工具/表單**：工作分配表、推車點檢表、KPI看板。

- **負責角色**：領班；(次)各區資深房務。

## 9. SOP清單(詳細步驟)
**SOP-01｜臭氧除臭**

1.  評估異味來源與範圍→記錄(房號/時間/來源判斷)。

2.  放置臭氧機中央位置，關閉門窗，設定［\_\_分］；門口放警示牌。

3.  完畢後全開窗/風口通風［\_\_分］；再次巡查確認無刺激味。

4.  系統紀錄：房態(處理中/可售)、「異味處理」欄位填寫、照片上傳。  
    **驗收**：二次巡檢合格；若仍有殘味→追加處置或報ENG檢修。  
    **KPI**：首次處置成功率≥［\_\_％］。

**SOP-02｜OOO管理(報修→驗收→釋房)**

1.  發現故障→立即報修(ENG)；標記OOO。

2.  協調完工時間；列入14:30–15:30驗收排程。

3.  驗收合格→改「髒房」並派清潔→改「空房」。  
    **驗收**：ENG＋HK雙簽；照片；系統時間戳。  
    **KPI**：驗收一次合格率≥［\_\_％］；解除OOO至可售≤［\_\_分］。

**SOP-03｜退房抽查與改空房**

1.  依抽查比例檢核關鍵點(擦塵、備品補充、床品規格、衛浴、設備抽測)。

2.  合格→改空房並通知FO；不合格→返工並再抽。  
    **驗收**：抽查表＋照片；PMS顯示可售。  
    **KPI**：抽查合格率≥［\_\_％］；退房至上架≤［\_\_分］。

**SOP-04｜DND關懷(階梯制)**

1.  首次發現→登錄時間；［\_\_h］後二次關懷(電話/門縫卡)。

2.  仍無回應且有風險→SEC陪同安全關懷。  
    **驗收**：關懷紀錄完整；必要時事件報告。  
    **KPI**：關懷準時率＝100%。

**SOP-05｜快速翻房(預進優先)**

1.  針對時間壓力房：重點清潔＋必需備品補足＋關鍵設備抽測。

2.  領班終檢合格→改空房→通知FO。  
    **驗收**：「快速翻房」標籤＋照片；時長記錄。  
    **KPI**：平均處理時長≤［**分］；客訴率≤［**％］。
