# 保全夜間巡檢路線與注意事項

## 巡檢時間表

| 次數 | 時間 | 重點區域 |
|------|------|---------|
| 第一巡 | 22:00 | 全棟外圍 + 1F 商場 |
| 第二巡 | 01:00 | B1F~B4F 停車場 + 機房 |
| 第三巡 | 04:00 | 各樓層走廊 + 逃生梯 |
| 第四巡 | 06:00 | 開店前商場全區 |

## 標準巡檢路線（第一巡）

```
大門廣場 → 1F 大廳 → 1F 商場（順時針）
→ B1F 停車場入口 → 機電房區域
→ 搭電梯至 RF 屋頂 → 逐層往下走逃生梯
→ 回到 1F 警衛室 填寫巡檢記錄
```

## 巡檢記錄方式
1. 登入 Portal → 保全巡檢 → 選擇對應班次
2. 逐項填寫各區域狀況（正常 / 異常）
3. 發現異常立即拍照，填入「異常說明」
4. 完成後點擊提交

## 緊急聯絡
- 消防 **119** / 警察 **110**
- 工務急線：**#2001**
- 值班主管：**#2010**

## 禁止事項
- ❌ 巡檢途中使用手機（緊急聯絡除外）
- ❌ 獨自進入封閉機房（需有第二人陪同）
- ❌ 發現可疑人物時自行處置（立即通報並等待支援）
