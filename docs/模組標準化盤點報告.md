## 2. 標準模組 hotel/daily-inspection 摘要

**檔案**: `C:\OneDrive\_Ragic\portal\frontend\src\pages\HotelDailyInspection\index.tsx`  
**路由**: `/hotel/daily-inspection`

### TAB 結構（共 7 個）
```
1. summary         (Dashboard 月份彙整 KPI + 月曆格)
2. daily-form      (每日巡檢表 - 標準月份 + 日期篩選)
3. rf              (RF 巡檢 - 月份篩選 + 場次清單)
4. 4f-10f          (4F~10F 巡檢 - 同上)
5. 4f              (4F 巡檢 - 同上)
6. 2f              (2F 巡檢 - 同上)
7. 1f              (1F 巡檢 - 同上)
```

### 5 大特徵

| 特徵 | 實現方式 | 說明 |
|------|---------|------|
| **Dashboard 篩選** | DatePicker (month picker) | `查詢月份：YYYY/MM` + 重新整理 + 同步全部 |
| **月曆呈現** | `MonthlyCalendarGrid` 元件 | 顯示各 Sheet 每日狀況（5 個巡檢 Sheet） |
| **TAB 結構** | 7 個 items，key = summary/daily-form/rf/4f-10f/4f/2f/1f | 懶載入 Tab 內容（openedTabs Set 追蹤） |
| **每日巡檢表 TAB** | `DailyInspectionFormTab` 子元件 | 月份 + DatePicker 篩選，呼叫 fetchHotelDailyBatches API |
| **TAB 篩選** | 各樓層 Tab: DatePicker (month) | 年月 + 重新整理 + 同步按鈕 |

### Dashboard KPI 卡片（5 張）
- 本月巡檢場次
- 已巡檢項目 (完成數 / 總數)
- 異常 + 待處理
- 整體完成率 %
- 本月巡檢時間 (小時)
